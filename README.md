# greeting-app
# 🎉 Greeting App

A simple and interactive Greeting App that displays personalized greetings based on user input. This project is beginner-friendly and perfect for learning basic UI development and logic building.

---

## 🚀 Features

- 👋 Displays greeting messages
- 🧑 Accepts user name input
- ⏰ Shows greetings based on time (Morning, Afternoon, Evening)
- 🎨 Simple and clean UI
- ⚡ Fast and responsive

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
